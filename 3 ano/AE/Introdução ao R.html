O que é o R?

  • Linguagem de Programação Estatística: focada em análise estatística e visualização de dados.
  • Software Livre e de Código Aberto: comunidade ativa e em desenvolvimento contínuo.
  • Ampla Gama de Pacotes: mais de 15 000 pacotes disponíveis no CRAN.
  • Usado em Ciência de Dados: ferramenta essencial para análise, aprendizagem automática e estatística.

Sintaxe Básica

  • Atribuição: <- ou =
    Exemplo:
x <- 10
x
[1] 10
  • Comentários: # Isto é um comentário
  • Funções básicas: print(), cat(), class(), str()
  • Operadores aritméticos: +, -, *, /, ^

Tipos de Dados

  • Numéricos: inteiros e reais (ex.: 1, 2.5)
  • Cadeias de caracteres: texto (ex.: "Olá Mundo")
  • Lógicos (booleanos): TRUE, FALSE
  • Fatores: categorias (níveis)
  • Datas e horas: Date, POSIXct, POSIXlt

Estruturas de Dados

  • Vetores: sequência de elementos do mesmo tipo
    Ex.: c(1, 2, 3, 4)
  • Matrizes: tabelas 2D de elementos do mesmo tipo
    Ex.: matrix(1:9, nrow = 3, ncol = 3)
  • Data Frames (Quadros de Dados): tabelas 2D com colunas de tipos distintos
    Ex.: data.frame(nome = c("Ana","Bruno"), idade = c(28,35))
  • Listas: coleções heterogéneas
    Ex.: list(vetor = 1:3, matriz = matrix(1:4,2), df = data.frame(x=1))

Operações Básicas com Vetores

vetor <- c(1, 2, 3, 4)
vetor * 2
[1] 2 4 6 8
vetor[1]        # primeiro elemento
[1] 1
vetor[2:4]      # do segundo ao quarto
[1] 2 3 4
length(vetor); sum(vetor); mean(vetor)
[1] 4
[1] 10
[1] 2.5

Estruturas de Controlo

Condicionais

if (x > 0) {
  print("Positivo")

Ciclos

for (i in 1:5) {
  print(i)
}

[1] 2
[1] 3
[1] 4

Visualização de Dados com ggplot2

# install.packages("ggplot2")
library(ggplot2)

ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point() +
  labs(title = "Dispersão wt vs mpg",
       x = "Peso (1000 lbs)", y = "Milhas/galão")
  • Estrutura básica: ggplot(data = dados, aes(x = var_x, y = var_y)) + geom_tipo()
  • Personalização: títulos, rótulos, temas.
